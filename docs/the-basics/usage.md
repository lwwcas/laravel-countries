# Basic Usage

## Usage

To import `Country::class`, use the following code:

```php{1}
use Lwwcas\LaravelCountries\Models\Country;

Country::all();

```

To import `CountryRegion::class`, use the following code:

```php{1}
use Lwwcas\LaravelCountries\Models\CountryRegion;

CountryRegion::all();

```
